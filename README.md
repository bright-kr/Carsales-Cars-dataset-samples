## Data Points

| Field Name | Description |
|------------|-------------|
| `car_id` | 각 자동차 리스팅에 대한 고유 식별자입니다. |
| `url` | 자동차 상세 리스팅 페이지로 바로 연결되는 URL입니다. |
| `manufacturer` | 자동차 제조사의 이름입니다. |
| `model` | 자동차의 구체적인 모델명입니다. |
| `badge` | 자동차 모델의 세부 변형 또는 트림입니다. |
| `year` | 자동차의 제조 연도입니다. |
| `price` | 자동차의 리스팅 가격입니다. |
| `currency` | 자동차 리스팅 가격의 통화입니다. |
| `main_image` | 자동차 리스팅의 대표 이미지 URL입니다. |
| `location` | 자동차가 제공되는 지리적 위치입니다. |
| `fuel_type` | 자동차가 사용하는 연료 유형입니다(예: petrol, diesel). |
| `seller_type` | dealer 또는 private 등 판매자 유형입니다. |
| `seller_id` | 자동차 판매자의 고유 식별자입니다. |
| `images` | 자동차의 추가 이미지 URL입니다. |
| `vehicle_description` | 차량의 특징 및 상태에 대한 설명입니다. |
| `powerplant_type` | 자동차를 구동하는 엔진 또는 모터의 유형입니다. |
| `cost_to_insure` | 자동차의 예상 보험 비용입니다. |
| `engine` | 자동차 엔진의 사양입니다. |
| `dealer_info` | 자동차 딜러에 대한 정보입니다. |
| `year_num` | 자동차 제조 연도의 숫자 표현입니다. |


## Use Cases

### 1. Market Price Analysis
이 데이터셋을 사용하여 다양한 자동차 모델 및 제조사 전반의 가격 추세를 분석할 수 있습니다. 이를 통해 기업은 시장 역학을 이해하고 경쟁력 있는 가격을 설정하는 데 도움을 받을 수 있습니다.

### 2. Automotive Market Research
연구자는 이 데이터셋을 활용하여 자동차 트렌드, 소비자 선호, 시간 경과에 따른 다양한 자동차 모델의 인기도를 연구할 수 있으며, 이는 전략적 의사결정에 도움이 됩니다.

### 3. Insurance Cost Estimation
보험사는 모델, 연도, powerplant type 등의 차량 속성을 기반으로 보험 비용을 추정하는 데 이 데이터를 사용할 수 있으며, 이를 통해 고객 요구에 맞춘 보험 상품을 구성할 수 있습니다.
